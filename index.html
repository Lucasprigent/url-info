site URL

definition url
Une URL, appelée adresse web, est une chaine de caractères uniforme qui permet d'identifier une ressource du World Wide Web par sont emplacement, de préciser le portocole Internet pour la récupérer,le nom de la machine,le chemin d'accès et le nom du fichier

signification URL
URL signifie uniform resource locator

creation de l'URL
En 1990 par Tim Berners-Lee, l'inventeur du Web met en ligne le premier site web de l'histoire sur l'URL : http://nxoc01.cern.ch/hypertext/WWW/TheProject.html. Ce site a ensuite évolué, car il a été régulièrement mis à jour.

Sa veux dire quoi

L'URL (Uniform Resource Locator) d'une page est l'adresse par laquelle un site est accessible. C'est cette adresse qui commence par « https » que vous écrivez dans la barre du navigateur Internet


De quoi se compose une URL ?

  1. Le protocole. C'est le fameux « http » (pour HyperText Transfer Protocol) que l'on retrouve au début de chaque adresse Internet. ...
  2. Le nom de domaine. C'est le serveur web avec lequel votre navigateur communique pour échanger du contenu. ...
  3. L'extension du nom de domaine. ...
  4. Le chemin d'accès.

Qu'est-ce que l'arborescence en informatique ?


L'arborescence est une structure hiérarchisée de données dans des répertoires et des sous-répertoires. L'image de l'arbre est une métaphore pour décrire l'organisation du chemin d'accès par branches et ramifications.


Comment chemine-t-on dans une arborescence ?


Il constitue le point de départ obligatoire pour accéder aux autres niveaux. Plus il y a de répertoires, plus on monte haut dans l'arbre: "C>Mes Documents>Mes Images>Photos>Mer".


Qu'est-ce qu'un chemin absolu ?


Le chemin absolu correspond à l'emplacement d'un répertoire ou d'un fichier sur un ordinateur ou sur le web. Il comprend le répertoire principal ainsi que tous les sous-répertoires. On utilise beaucoup les chemins dans les systèmes d'exploitation pour représenter les liens entre les fichiers et les répertoires.


Qu'est-ce qu'un chemin relatif ?


Une URL relative indique soit le chemin dans cette arborescence entre la page dans laquelle on l'écrit (la page courante) et la ressource qu'elle cible, soit le chemin dans cette arborescence entre la racine du site et cette ressource.


Comment accéder au fichier index.html d'un site web ?


Dans le menu Fichier, cliquez sur Ouvrir. Parcourez l'emplacement oÙ vous avez enregistré votre article, dans la rubrique « Création de votre document HTML » de cet article. Sélectionnez le fichier, puis cliquez sur Ouvrir.


Peut-on avoir une IP dans l'URL ?


Les adresse IP sont trouvées à partir d'une url grâce aux serveurs DNS pour Domain Name Server. On dit qu'un nom DNS est résolu grâce à un serveur DNS. Il y a des milliers de serveurs DNS dans le monde et ils se synchronisent continuellement.
